    34. Reinoud de Lange on October 17

      I got the IDE to work on my mac:
      I changed the file structure to this: http://dlanger.home.xs4all.nl/change-file-structure-arduino-microduino.png
      I copied the file platform.txt from ../arduino/avr/ to ../Microduino/avr/
      and added the following lines to the boards.txt:

      32u416m.upload.tool=avrdude

      32u416m.upload.use_1200bps_touch=true
      32u416m.upload.wait_for_upload_port=true

      32u416m.build.usb_product="Microduino-Core USB"
      32u416m.build.usb_manufacturer="Microduino"
      32u416m.build.extra_flags={build.usb_flags}

      Then I started the IDE and it worked!
